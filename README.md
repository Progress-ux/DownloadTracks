## Способы установки зависимостей
1. 
```bash
pip install -r requirements.txt
```
2. 
```bash
pip install yt-dlp requests mutagen tqdm
```

## Запуск

```bash
python main.py
```

## Использование
### Для использования скрипта соберите ссылки формата: 
- `https://youtu.be/{id-video}`
### Ссылки такого формата не поддерживаются:
- `https://youtu.be/{id-video}?list={id-list}`

## Файл track.txt
Он служит для удобного копирования ссылок. Он никак не взаимодействует со скриптом
