# Youtube Downloader MP3

## Создание venn

```bash
python -m venv venv
```

```bash
source ./venv/bin/activate
```

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python qt_main.py
```

## Использование

### Для использования скрипта соберите ссылки формата: 

- `https://youtu.be/{id-video}`
  
### Ссылки такого формата не поддерживаются:

- `https://youtu.be/{id-video}?list={id-list}`
